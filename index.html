      📚 Base de plats

      💡 Comment fonctionne la sauvegarde ?

      Quand tu ajoutes ou modifies un plat, c'est enregistré dans ce navigateur (mémoire locale). C'est rapide, mais fragile : si tu vides le cache ou ouvres l'outil sur un autre appareil, les ajouts disparaissent.

      Pour ne rien perdre, tu as 3 options :
      1. Sauvegarde automatique JSON — à chaque ajout, un fichier base-plats-neuils-backup.json est téléchargé dans tes Téléchargements. Garde le dernier.
      2. Dossier OneDrive — configure un dossier ci-dessous, la sauvegarde s'y écrit directement (Chrome/Edge PC uniquement). OneDrive synchronise ensuite.
      3. ✅ La vraie solution durable : "Sauvegarder dans l'outil" — génère un nouveau index.html avec tes plats gravés dedans. Remplace le fichier sur GitHub → permanant sur tous les appareils, pour tout le monde.

      Chrome/Edge PC uniquement. Le dossier choisi est mémorisé, pas besoin de le resélectionner à chaque fois.



      Génère un index.html avec tous tes plats gravés dans le code. À uploader sur GitHub.
